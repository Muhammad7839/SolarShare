Frequently Asked Questions

Do customers need rooftop installation?

No. SolarShare is built around shared/community solar participation through utility-linked credits.

Is the recommendation model transparent?

Yes. Inputs, ranking priorities, and data-source status are displayed directly in the workflow.

What if live sources are unavailable?

The app switches to fallback-safe mode and labels the context so users still receive valid outputs.
